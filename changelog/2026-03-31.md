# Changelog: March 31, 2026

## 🚀 New Features
- **Join Group Acceptance Flow**: Users can now formally join groups via `POST /groups/{id}/join`. The operation is atomic, updating both the group profile and user membership records simultaneously.
- **Real-time WebSocket Notifications**: Bridge the Phase 4 gap with end-to-end WebSocket support.
  - Added `WsHandlerFunction` for session management ($connect/$disconnect).
  - Integrated `useWebSocket` hook in React frontend.
  - Automatic dashboard refresh when `RECEIPT_SCAN_COMPLETED` is received.

## 🛠️ Bug Fixes & Refinement
- **LocalStack Integration Parity**: 
  - Resolved 404/500 routing issues in LocalStack using a prefix-agnostic keyword matching strategy in `normalizeRoute`.
  - Fixed partition key mismatch where group expenses were being queried from the wrong DynamoDB partition.
  - Added partition-agnostic fallback (Scan) for local development `GET`/`PATCH` operations on expenses.
- **CDK Infrastructure**: Added missing routes for `/settle` and `/join` to `CostsCrunchStack`.

## 🧪 Testing
- Verified **Groups & Settlements** integration suite in LocalStack (100% pass).
- Added and verified **Expenses HTTP Export** integration test.
